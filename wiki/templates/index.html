Welcome to Propolis Wiki

Propolis is a wiki which utilizes the Hive blockchain for data storage. The name comes from the Greek word propolis (pro ‘before’ + polis ‘city’), also used for the bee glue that keeps the hive together. This suitably describes the intention of the project as an entry point to the community which connects all areas.

Scope

Articles can cover all kinds of topics, but naturally we start with everything Hive. Ideally this will be the central place to get all information about your favorite dApp, game or wallet.

Explore

If you're not sure where to start, how about a random article? You can also browse the existing categories under which articles have been published, or see what's happened lately on the activity list!

Contribute

Everyone can contribute to Propolis, the only requirement is a Hive account. For now, posting rights are managed manually - join our Discord to get you set up! {% endblock %}
